Practice SQL queries with real-world datasets.

## 🏥 Dataset: Hospital Data

📂 Dataset Features:

🏥 Hospital Name

📍 Location

🏨 Departments

👨‍⚕️ Number of Doctors

👩‍⚕️ Number of Patients

🏥 Admission & Discharge Dates

💰 Medical Expenses
